Tech Fest 2025: Web Development Challenge

Round 1 - Medium Level Questions

Instructions

Welcome to Round 1 of the Web Development event! Solve the 4 questions below to qualify for Round 2. Submit your solutions as working code (HTML, CSS, JavaScript) via the provided submission link. Time limit: 2 hours. Use only standard web technologies (no frameworks unless specified). Good luck!

Question 1: Responsive Navigation Bar

Create a responsive navigation bar using HTML and CSS. The navbar should:

  • Display horizontally on desktop (width > 768px) with 4 links: Home, About, Services, Contact.
  • Collapse into a hamburger menu on mobile (width ≤ 768px) that toggles to show links vertically.
  • Use CSS only for the toggle functionality (no JavaScript).

Output: A working navbar with responsive behavior.

Question 2: Form Validation with JavaScript

Build a registration form with fields for username, email, and password. Validate using JavaScript:

  • Username: Minimum 5 characters.
  • Email: Must contain "@" and a domain (e.g., ".com").
  • Password: Minimum 8 characters, at least one number.
  • Show error messages below each field if validation fails.
  • Display a success message if all inputs are valid.

Output: A functional form with validation and messages.

Question 3: Dynamic To-Do List

Create a to-do list app using HTML, CSS, and JavaScript. Features:

  • Add tasks via an input field.
  • Mark tasks as complete (e.g., strikethrough or checkbox).
  • Delete tasks.
  • Store tasks in localStorage to persist after page refresh.

Output: A working to-do list with all features.

Question 4: CSS Grid Photo Gallery

Design a photo gallery with 6 images using CSS Grid. Requirements:

  • Desktop (width > 1024px): 3x2 grid.
  • Tablet (768px-1024px): 2x3 grid.
  • Mobile (width ≤ 768px): Stack images vertically.
  • Ensure images scale without distortion.

Output: A responsive gallery with correct layouts.
